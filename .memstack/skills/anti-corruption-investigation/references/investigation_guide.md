# 反腐调查参考指南

## 调查流程

### 1. 初步评估阶段
- 数据收集和整理
- 快速风险筛查
- 确定调查范围

### 2. 深度分析阶段
- 关键词分析
- 通信模式识别
- 异常行为检测
- 时间线重构

### 3. 证据收集阶段
- 截图和录屏
- 日志保存
- 证人访谈
- 物证收集

### 4. 报告生成阶段
- 汇总分析结果
- 风险等级评定
- 提出处理建议
- 形成正式报告

## 可疑模式识别

### 资金相关
- 大额转账讨论
- 现金交易约定
- 回扣、佣金提及
- 贿赂相关词汇

### 行为异常
- 频繁私下会面
- 非工作时间联系
- 要求保密或删除记录
- 使用暗语或加密

### 权力滥用
- 违规办事请求
- 特殊照顾承诺
- 利益输送暗示
- 关系网运作

## 风险等级标准

### 高风险 (红色)
- 多次出现直接腐败词汇
- 明确的金钱交易约定
- 销毁证据的尝试
- 高风险用户评分 > 平均值2倍

### 中风险 (黄色)
- 偶尔出现可疑词汇
- 异常通信模式
- 非工作时间频繁联系
- 风险评分 3-5分

### 低风险 (绿色)
- 可疑词汇很少
- 正常通信模式
- 无明显异常行为
- 风险评分 < 3分

## 法律依据

### 相关法律法规
- 刑法第383条: 贪污罪
- 刑法第385条: 受贿罪
- 刑法第389条: 行贿罪

### 证据要求
- 真实性
- 合法性
- 关联性
- 充分性

## 注意事项

1. **数据隐私**: 严格遵守数据保护法规
2. **证据保全**: 确保数据完整性和不可篡改
3. **程序合法**: 调查过程符合法律程序
4. **保密原则**: 调查信息严格保密
5. **客观公正**: 基于事实，避免主观臆断

## 工具使用
